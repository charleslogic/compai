    How this works & notes
    • Clipboard: your question is copied to the clipboard when you click Search. If your browser blocks automatic copying (common when opening this file directly from disk), the question is shown in the status message below so you can copy it manually.
    • Pop-up blockers: browsers may block more than one or two automatically-opened tabs per click. If a tab doesn't open, allow pop-ups for this page, or click the "Open" link for that service in the results list (a manual click is never blocked).
    • Pre-filling the question: only Perplexity is known to reliably accept a question via its URL and run the search automatically (green badge). ChatGPT, Claude, Microsoft Copilot, and You.com may pre-fill the input box via an undocumented URL parameter, but this isn't guaranteed and can change at any time (blue badge). For all other services - and as a fallback for the rest - paste (Ctrl+V / Cmd+V) your question and press Enter (gray badge).
    • Your checkbox selections are remembered on this device for next time.
